## 📌 Project Overview
This project focuses on implementing the **Minimax algorithm with Alpha-Beta pruning** to create an intelligent agent for Connect Four. The goal is to optimize adversarial search by reducing the number of nodes explored in the game tree without sacrificing decision quality.

## 🛠️ Technical Implementation
* **Search Algorithm:** Recursive Minimax.
* **Optimization:** Alpha-Beta Pruning ($\alpha-\beta$) to improve computational efficiency.
* **Evaluation Function:** A heuristic utility function $U(s)$ to estimate board strength at non-terminal states.
* **Analysis:** Comparative study of nodes expanded and runtime between standard and optimized search.
