📌 Project Overview
This project implements the Minimax algorithm with Alpha-Beta pruning to create an intelligent agent for a two-player, zero-sum board game (e.g., Connect Four). While Minimax guarantees optimal decision-making, its exponential time complexity is a bottleneck. By incorporating Alpha-Beta pruning, we drastically reduce the state-space search without compromising the quality of the move.

🛠️ Technical Features
Adversarial Search: Implementation of the Minimax algorithm.

Optimization: Alpha-Beta pruning to eliminate unnecessary branches.

Heuristic Evaluation: A custom utility function to estimate board strength at non-terminal states.

Performance Metrics: Comparison of nodes expanded and runtime between standard Minimax and optimized Alpha-Beta.
